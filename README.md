# Face Distance Calculator

A professional Streamlit web application for calculating face distances and performing identity verification. This application is designed for use in medical, construction, and corporate environments where secure identity verification is required.

## Features

- **Dual Image Upload**: Upload reference and comparison images
- **Face Distance Calculation**: Calculate Euclidean distance between face encodings
- **Professional Interface**: Clean, business-appropriate design
- **Real-time Processing**: Fast face recognition and comparison
- **Comprehensive Results**: Detailed analysis with confidence scores
- **Error Handling**: Robust validation and error reporting
- **Mobile Responsive**: Works on desktop and mobile devices

## Use Cases

- ✅ Patient identity check in clinics
- 🏗️ Worker verification on construction sites
- 🏢 Visitor access validation in corporate buildings

## Quick Start

1. **Install Dependencies**
   ```bash
   pip install streamlit pillow numpy
   ```

2. **Run the Application**
   ```bash
   streamlit run app.py --server.port 5000
   ```

3. **Access the Application**
   Open your browser and navigate to `http://localhost:5000`
